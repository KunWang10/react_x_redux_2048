# react_x_redux_2048
# How to compile
Clone the repo:
```
git clone git@github.com:KunWang10/react_x_redux_2048.git
```
Enter the folder:
```
cd react_x_redux_2048
```
Run:
```
yarn
```
to install dependencies;

Then you run:
```
yarn start
```
The browser will open automatically, or you can manually open [localhost:3000](localhost:3000).
